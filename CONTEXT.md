# Training Philosophy

Program is designed for hypertrophy with progressive overload.

Main principles:

- Progress reps before increasing weight.
- Use clean technique.
- Compounds:
  - Usually stop with 1–2 RIR.
- Isolation:
  - Can be taken close to failure.
- Do NOT test max strength during working sets.
- Working weights should remain consistent across sets whenever possible.
- Avoid unnecessary weight changes every set.
- Small improvements every week are preferred over large jumps.
